# IronLog

**MVP** фитнес-трекера на **React + TypeScript + Vite**. Один экран — одна зона контента (до ~430px), нижняя навигация, все данные тренировок и статистики в **localStorage**.

## Что построили за один вечер

- **Библиотека упражнений** с поиском и фильтрами по группам мышц
- **Карточка упражнения** с техникой, мышцами, тегами и добавлением в корзину
- **Создание своих упражнений** (bottom sheet, сохранение в `ironlog_custom_exercises`, бейдж «Моё», удаление на карточке)
- **Конструктор тренировки** с корзиной (**WorkoutContext**) и **суперсетами**
- **Режим выполнения** с таймером тренировки, **таймером отдыха** между сетами и прогрессом по шагам
- **Редактирование выполненных сетов** на активной тренировке и на экране проверки перед сохранением
- **Экран завершения** с итогами, проверкой результатов и **новыми PR**
- **Дашборд прогресса**: стрик, метрики, общий тоннаж и вехи, график 7 дней, топ PR, **ачивки** и toast при разблокировке
- **Сохранение в localStorage** (программы, логи, PR, кастомные упражнения, незавершённая сессия)

## Ключи localStorage

| Ключ | Назначение |
|------|------------|
| `ironlog_workouts` | Сохранённые программы |
| `ironlog_logs` | Логи завершённых тренировок |
| `ironlog_prs` | Личные рекорды |
| `ironlog_custom_exercises` | Свои упражнения |
| `ironlog_active_workout` | Текущая сессия до сохранения |

## Команды

```bash
npm run build
```
